### Toward Intelligent Circulation  

#### Modeling Collapse, Inefficiency, and System Optimization

This project develops a mathematical and computational framework to analyze traffic system instability, inefficiency, and collapse dynamics.

Core areas include:

- System stability and perturbation growth  
- Congestion modeling and collapse thresholds  
- Lagrangian-inspired system analysis  
- Distributed vs. centralized optimization  
- Integration with sensing technologies (e.g., LiDAR)
